# BAS Graphic Generator v1

Professional HVAC Controls / BAS graphics generator built with:

- Python
- Flask
- OpenCV
- SVG Export
- PDF Processing
- Smart Cleanup Rendering

## Features

✅ PNG / JPG Upload  
✅ PDF Upload  
✅ Transparent PNG Export  
✅ SVG Vector Export  
✅ Zoom / Pan Viewer  
✅ Auto Rotate Plans  
✅ Smart Cleanup Mode  
✅ Synchrony Style Graphics
